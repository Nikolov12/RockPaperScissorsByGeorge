# RockPaperScissorsByGeorge
This is a simple console game 'Rock Paper Scissors' based Python.

<img width="300" height="275" alt="Rock-paper-scissors svg" src="https://github.com/user-attachments/assets/58303354-b188-473b-8bdb-b72ae49dfa97" />

Rock, Paper, Scissors is a simple hand game for two players where each person simultaneously forms one of three shapes (rock, paper, or scissors) to beat their opponent's choice, following the rule: rock crushes scissors, scissors cuts paper, and paper covers rock. Players usually chant "Rock, Paper, Scissors, Shoot!" while making fist movements and reveal their choice on "Shoot," with ties leading to a replay. 

# Input and Output

The player enters one of the following options:
- r - rock
- p - paper
- s - scissors

The computer chooses random option, the reveals the winner.
